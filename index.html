Luma — full USB flash

Use this page on a desktop with Chrome or Edge (Web Serial). Connect your ESP32-S3 board with a data-capable USB cable.

  1. Put the device in download mode (typical DevKit: hold BOOT, tap RESET, release BOOT). If the board looks “bricked,” this still works: the ROM bootloader in mask ROM is intact—only the flash bootloader or app may be wrong.
  2. Click the button below and choose the serial port.
  3. Allow a full erase when asked if this is a new install or recovery.
Use Chrome or Edge on a desktop. Web Serial is not available in this browser. Open this page over HTTPS (required for Web Serial).

After flashing, the device can use normal Wi‑Fi / OTA updates. This page is only for hardware that never had OTA or needs a clean recovery.
